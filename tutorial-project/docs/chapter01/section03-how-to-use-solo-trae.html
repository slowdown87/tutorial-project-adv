1.3 如何获取和使用Solo Trae Web Code

目标

在本节中，你将：

  • 了解如何获取Solo Trae Web Code
  • 学习如何注册账号
  • 知道如何打开和登录
  • 认识界面的每个部分
  • 学会基本的操作方法
  • 完成第一次与AI的对话

    • ---

    第一步：获取Solo Trae Web Code

    在使用之前，你需要先获取Solo Trae Web Code。

    在哪里找到？

    Solo Trae Web Code通常可以通过以下方式获取：

    #### 方式1：通过官方网站（最常见）

    1. **打开浏览器**

    - 找到你电脑上的浏览器图标（Chrome、Firefox、Edge等）

    - 双击图标打开浏览器

    2. **访问官方网站**

    - 在浏览器顶部的地址栏中，输入：`https://solo.trae.cn/`

    - 按回车键（Enter键）

    3. **查看首页**

    - 你会看到Solo Trae Web Code的官方主页

    - 上面应该有"开始使用"、"注册"或类似的按钮

    ---

    #### 方式2：通过其他平台

    有时候，Solo Trae Web Code也可能在其他平台上提供：

  • 如果是某个IDE的插件：在你的代码编辑器的插件商店中搜索
  • 如果是桌面应用：在应用商店或官网下载安装包
  • 如果是通过学校/公司：询问相关负责人获取入口

    • ---

    第二步：获取邀请码并注册账号

    **重要提示**：目前Solo Trae Web Code处于内测阶段，需要邀请码才能使用！

    方式一：通过官网申请（推荐）

    1. **访问申请页面**

    - 打开浏览器，访问：`https://trae.com.cn/`

    - 这是TRAE中国官网，最正规的渠道

    2. **找到申请入口**

    - 在首页上找到"立即探索SOLO"或"预约申请"按钮

    - 点击这个按钮

    3. **提交预约信息**

    - 在申请页面填写你的手机号码

    - 点击"立即预约"按钮

    - 你会看到"提交成功！我们将及时为您更新信息"的提示

    4. **等待官方通知**

    - 目前每天开通200个资格

    - 每天下午18:00截止当天申请

    - 按照提交顺序逐批开通

    - 请耐心等待官方短信或邮件通知

    ---

    方式二：通过官方直播获取

    1. **关注官方直播间**

    - 官方会在以下平台直播：

    - 微信视频号

    - 抖音

    - B站

    - 小红书

    2. **观看直播**

    - 在指定时间观看直播（通常是晚上）

    - 直播过程中会有多轮邀请码发放

    3. **参与互动**

    - 通过抽奖、答题等互动方式

    - 有机会获得邀请码

    ---

    方式三：通过社区活动获取

    关注TRAE官方账号和社区：

  • 官方论坛：`https://forum.trae.cn/`
  • 官方B站账号：TRAE_ai
  • 后续会不定期通过活动发放邀请码

    • ---

    注册并登录（获得邀请码后）

    获得邀请码后，按照以下步骤操作：

    1. **打开SOLO网页版**

    - 访问：`https://solo.trae.cn/`

    2. **注册账号**

    - 使用手机号注册

    - 输入短信验证码

    - 输入你的邀请码激活

    3. **开始使用**

    - 登录成功后，就可以开始使用了！

    ---

    第三步：打开并进入工作区

    登录成功后，你就可以开始使用了！

    进入工作区

    1. **查看主页**

    - 登录后，你会看到用户主页

    - 可能有"项目"、"工作区"或类似的选项

    2. **创建或打开项目**

    - 如果是第一次使用，可能需要创建一个新项目

    - 点击"新建项目"或"创建项目"按钮

    - 或者，如果已经有项目了，直接点击打开

    3. **进入编辑界面**

    - 现在你应该能看到和本教程类似的界面了！

    - 这就是我们的工作区

    ---

    第四步：认识界面布局

    恭喜你！现在你已经打开了Solo Trae Web Code，让我们来认识一下界面的各个部分。

    让我们从左到右来看：

    #### 📁 区域1：左侧 - 文件管理区

    **位置**：界面的最左边

    **样子**：看起来像一个文件夹列表

    **这里能做什么**：

  • 👀 查看你的项目有哪些文件
  • 📂 看到文件夹和文件的层级结构
  • 📝 双击文件名可以打开并编辑文件
  • ➕ 右键点击可以创建新文件或文件夹
  • 🗑️ 右键点击也可以删除不需要的文件

    • **现在试试看**：

    看向你的左侧，你能看到什么？

  • 可能有一些文件夹和文件
  • 这就是我们的项目文件！

    • ---

    #### 💬 区域2：中间 - 对话区（最重要的区域！）

    **位置**：界面的中间部分

    **样子**：看起来像一个聊天窗口

    **这里能做什么**：

  • ✏️ 在底部的输入框中打字
  • 📤 按回车或点击"发送"按钮，把你的消息发给AI
  • 📖 查看AI给你的回复
  • 🔙 往上滚动，可以看到之前的对话历史

    • **这是你和AI交流的地方！** 有什么想法、问题、需求，都在这里说。

    ---

    #### 🏠 区域3：顶部 - 功能栏

    **位置**：界面的最上方

    **样子**：一排按钮或图标

    **常见的按钮**：

  • 🏠 "主页"或"项目"：切换到其他项目
  • 📊 "任务"：查看正在进行的任务
  • ⚙️ "设置"：调整一些选项
  • 🔔 "通知"：查看消息提醒
  • 👤 用户头像：查看个人信息或退出登录

    • **现在不需要记住所有按钮**，用到的时候我们再细说！

    ---

    #### 👁️ 区域4：右侧 - 预览区（可能有）

    **位置**：界面的最右边（不是所有界面都有）

    **可能的样子**：

  • 代码预览窗口
  • 网页预览窗口
  • 或者其他工具

    • **如果有的话**，我们可以在这里看到代码运行的效果！

    第五步：你的第一次对话（实战！）

    现在让我们来实际操作一下，试试和AI说话！

    操作1：输入你的第一句话

    1. **找到输入框**

    - 看对话区的底部

    - 有一个空白的框，可以在里面打字

    2. **点击输入框**

    - 用鼠标点击一下这个空白框

    - 你会看到光标在里面闪烁

    3. **输入一句话**

    - 试着输入："你好！"

    - 或者输入："我想学习编程"

    4. **发送消息**

    - 按一下键盘上的回车键（Enter键）

    - 或者点击输入框旁边的"发送"按钮

    5. **等待AI回复**

    - 稍等几秒钟

    - AI就会回复你了！

    ---

    操作2：查看文件

    1. **看向左侧的文件区**

    - 看看有哪些文件夹和文件

    - 如果文件夹前面有一个小箭头，点击它展开

    2. **找到一个文件**

    - 找一个你感兴趣的文件

    - 双击文件名

    - 中间区域可能会显示这个文件的内容

    3. **看看里面写了什么**

    - 这就是项目文件！

    ---

    操作3：创建一个简单的文本文件（可选）

    1. **在文件区右键点击**

    - 在空白处或者在文件夹上点击右键

    - 会弹出一个菜单

    2. **选择"新建文件"**

    - 在菜单中找到"新建文件"或"New File"

    - 点击它

    3. **输入文件名**

    - 输入：`test.txt`

    - 按回车键确认

    4. **编辑文件**

    - 双击刚创建的 `test.txt`

    - 在里面输入一些文字，比如："这是我的第一个文件！"

    5. **保存文件**

    - 通常会自动保存

    - 如果没有，按 `Ctrl+S`（Windows）或 `Cmd+S`（Mac）

    恭喜！你已经创建了你的第一个文件！

    常见问题解答

    Q: 我不知道在哪里获取Solo Trae Web Code怎么办？

    A: 你是从哪里看到这个教程的？那个地方应该会告诉你在哪里获取。如果是在网上看到的，可能需要搜索一下"Solo Trae Web Code"找到官网。

    Q: 注册时遇到问题怎么办？

    A: 别着急！通常网站上会有"帮助"或"常见问题"页面，你可以去那里看看。如果还是不行，可以找客服或技术支持。

    Q: 我找不到某个功能怎么办？

    A: 没关系！在本教程中，我会一步步带你找到需要的功能。如果真的找不到，可以直接在对话区问AI："我找不到XX功能，在哪里？"

    Q: 我操作错了，把文件删了怎么办？

    A: 别着急！如果是刚删除的，很多时候可以撤销。如果找不到了，也没关系，我们可以重新创建。在对话区告诉AI："我不小心删除了XX文件，能帮我恢复吗？"

    Q: 界面和你说的不一样怎么办？

    A: 界面可能会有更新，或者不同的版本看起来略有不同。但大体功能是一样的！你可以描述一下你看到的界面，AI会帮你确认。

    Q: 我可以用中文和AI交流吗？

    A: 当然可以！Solo Trae Web Code完全支持中文，你尽管用中文和AI交流！

    Q: AI回复得慢怎么办？

    A: 有时候AI需要一点时间思考，特别是复杂的问题。请稍等一会儿，如果等待时间太长，可以试试重新发送，或者把问题拆分成几个小问题。
