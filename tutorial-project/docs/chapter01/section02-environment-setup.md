# 1.2 环境准备

## 目标

在本节中，你将：
- 了解完成本教程需要准备哪些工具
- 确认你的环境是否符合要求
- 知道接下来需要做什么

---

## 所需工具清单

要完成本教程，你需要准备以下工具：

### 1. 一台能上网的电脑

这是最基本的要求！你需要：
- 💻 一台电脑（Windows、Mac或Linux都可以）
- 🌐 稳定的网络连接

---

### 2. Solo Trae Web Code

这是我们的主要工具！我们将用它来：
- 📝 编写代码
- 📁 管理项目文件
- 💬 和AI助手交流
- 📚 生成教程内容

**好消息是**：
- ✅ 不需要下载安装
- ✅ 不需要复杂的配置
- ✅ 打开就能用

**在第1.3节中**，我们会详细教你如何打开和使用它！

---

### 3. 浏览器

你需要一个现代的浏览器来：
- 打开Solo Trae Web Code
- 访问网站
- 预览我们的项目
- 注册各种服务账号

**推荐的浏览器**：
- 🌐 Google Chrome（推荐）
- 🦊 Mozilla Firefox
- 🧭 Microsoft Edge
- 🍎 Safari（Mac用户）

**如何确认你有浏览器？**
- 看看你的电脑桌面上，有没有上面提到的浏览器图标
- 如果没有，你需要先安装一个（通常电脑都自带浏览器）

---

### 4. Gitee账号（免费）

Gitee是我们选择的国内Git托管平台。

**为什么选择Gitee？**
- 🚀 国内访问速度快
- 🆓 完全免费
- 🇨🇳 中文界面友好
- 📚 功能全面

**如何注册？** 别担心，我们在[第4章](../chapter04/section02-cnb-intro.html)会详细讲解！

现在你只需要知道：我们需要一个cnb.cool账号就可以了。

---

### 5. cnb.cool账号（免费）

cnb.cool是我们选择的一站式代码托管和静态网站托管服务。

**为什么选择cnb.cool？**
- 🏠 完全国内服务，访问速度快
- 🆓 完全免费
- ⚡ 速度快且稳定
- 🔒 一站式服务，代码托管和网站部署在一起

**如何注册？** 同样，我们在[第4章](../chapter04/section03-cnb-register-repo.html)会详细讲解！

现在你只需要知道：我们需要一个cnb.cool账号就可以了。

---

## 检查清单

在继续之前，请确认你已经准备好：

- [ ] 有一台能上网的电脑
- [ ] 电脑上安装了现代浏览器（Chrome、Firefox等）
- [ ] 准备好了要注册Gitee账号
- [ ] 准备好了要注册Cloudflare账号

**不用担心Solo Trae Web Code**，我们在下一节会教你如何打开它！

## 不需要的工具

**好消息！** 以下工具你**不需要**安装：

- ❌ 复杂的IDE（如Visual Studio）
- ❌ 本地服务器环境
- ❌ 数据库
- ❌ 其他复杂的开发工具

Solo Trae Web Code会帮我们处理这些复杂的事情！
