# 如何让新对话的AI采用主动引导对话方式

## 问题背景

当用户开始一个新的对话时，如何让AI能够主动引导对话，而不是被动地等待用户提供所有信息？这是许多用户在与AI协作时面临的挑战。

## 核心解决方案

要让新对话的AI采用主动引导的对话方式，主要有以下几种方法：

### 方法1：使用系统提示（System Prompt）

**原理**：通过在对话开始前设置系统提示，告诉AI应该采用主动引导的对话风格。

**具体步骤**：

1. **在新对话开始时**，在输入框中输入以下系统提示：

```
请你作为一个主动引导型的AI助手，采用结构化的方式帮助我完善需求。

当我提出初始需求时，请按照以下流程与我交流：

1. 阶段1：初始需求识别
   - 识别核心需求和目标
   - 评估信息的完整性
   - 主动提出引导性问题

2. 阶段2：需求深化与细化
   - 分析我的回答
   - 识别信息缺口
   - 提出更具体的问题
   - 提供初步的方案框架

3. 阶段3：方案制定与确认
   - 整合所有信息
   - 制定完整的方案
   - 呈现方案并请求确认
   - 提供实施路径

4. 阶段4：执行与反馈
   - 定期检查进度
   - 主动询问遇到的问题
   - 提供具体的建议和解决方案
   - 根据反馈调整方案

请使用清晰的结构和层次分明的提问方式，帮助我系统地思考和完善需求。
```

2. **然后**，在同一消息中提出你的具体需求。

**示例**：

```
请你作为一个主动引导型的AI助手，采用结构化的方式帮助我完善需求。

当我提出初始需求时，请按照以下流程与我交流：

1. 阶段1：初始需求识别
   - 识别核心需求和目标
   - 评估信息的完整性
   - 主动提出引导性问题

2. 阶段2：需求深化与细化
   - 分析我的回答
   - 识别信息缺口
   - 提出更具体的问题
   - 提供初步的方案框架

3. 阶段3：方案制定与确认
   - 整合所有信息
   - 制定完整的方案
   - 呈现方案并请求确认
   - 提供实施路径

4. 阶段4：执行与反馈
   - 定期检查进度
   - 主动询问遇到的问题
   - 提供具体的建议和解决方案
   - 根据反馈调整方案

请使用清晰的结构和层次分明的提问方式，帮助我系统地思考和完善需求。

我的需求是：我想做一个个人网站，展示我的作品集。
```

### 方法2：使用简短的引导指令

**原理**：如果不想使用长系统提示，可以使用简短的引导指令来提醒AI采用主动引导的方式。

**具体步骤**：

1. **在新对话开始时**，使用以下格式提出需求：

```
【主动引导模式】我的需求是：[你的具体需求]

请你主动引导我完善需求，提出系统性的问题，帮助我深入思考项目细节。
```

**示例**：

```
【主动引导模式】我的需求是：我想学习Python编程

请你主动引导我完善需求，提出系统性的问题，帮助我深入思考学习计划的细节。
```

### 方法3：使用模板化的初始提问

**原理**：直接使用我们之前创建的AI主动引导对话模板中的提问结构。

**具体步骤**：

1. **在新对话开始时**，参考[AI主动引导对话模板](file:///workspace/tutorial-project/docs/appendix/ai-active-guidance.md)中的示例，提出一个包含结构化问题的初始消息。

**示例**：

```
我想做一个移动应用，帮助用户管理日常任务。

为了让你能更好地帮助我，请你按照以下结构主动引导我完善需求：

【项目背景】
- 这个应用的目标用户是谁？
- 你希望应用解决什么具体问题？

【具体功能】
- 应用需要包含哪些核心功能？
- 有没有特定的设计风格或参考应用？

【技术要求】
- 你倾向于使用什么技术栈开发？
- 有没有平台限制？（iOS、Android、跨平台等）

【期望结果】
- 你希望通过这个应用达到什么目标？
- 有没有时间或预算限制？
```

## 实际应用场景

### 场景1：新项目启动

**用户初始消息**：

```
【主动引导模式】我想创建一个在线教育平台

请你主动引导我完善需求，提出系统性的问题，帮助我深入思考项目细节。
```

**AI响应**：

```
很高兴你想创建一个在线教育平台！为了给你提供最全面的方案，我需要了解一些关键信息：

【项目背景】
- 这个平台的目标受众是谁？（学生、专业人士、兴趣爱好者等）
- 平台将提供哪些类型的课程？（技术、艺术、语言等）

【核心功能】
- 平台需要包含哪些基本功能？（课程管理、用户管理、支付系统等）
- 有没有特色功能需求？（互动学习、证书系统等）

【技术选择】
- 你倾向于使用什么技术栈？
- 有没有预算或技术团队的限制？

【运营规划】
- 你计划如何获取用户？
- 课程内容将如何生产和管理？

有了这些信息，我就能为你制定一个详细的在线教育平台开发方案！
```

### 场景2：问题解决

**用户初始消息**：

```
【主动引导模式】我的网站流量下降了，不知道怎么办

请你主动引导我分析问题，提出系统性的解决方案。
```

**AI响应**：

```
网站流量下降确实是一个需要重视的问题！让我帮你分析和解决。为了提供针对性的建议，我需要了解一些具体情况：

【问题描述】
- 流量下降的程度如何？（下降了多少百分比）
- 这个问题是从什么时候开始的？
- 是所有页面流量都下降还是特定页面？

【网站情况】
- 网站的主要内容和目标是什么？
- 最近有没有做过网站改版或内容调整？
- 有没有使用分析工具监测流量？

【外部因素】
- 行业内是否有重大变化？
- 竞争对手的情况如何？
- 有没有受到搜索引擎算法更新的影响？

有了这些信息，我就能为你分析流量下降的原因并提供具体的解决方案！
```

### 场景3：学习规划

**用户初始消息**：

```
【主动引导模式】我想学习数据分析

请你主动引导我制定学习计划，帮助我明确学习路径和目标。
```

**AI响应**：

```
非常好！数据分析是一个很有价值的技能。为了给你制定最适合的学习计划，我需要了解：

【学习背景】
- 你目前的数学和编程基础如何？
- 学习数据分析的目的是什么？（职业发展、个人兴趣等）
- 每天能投入多少时间学习？

【学习目标】
- 你希望达到什么水平？（入门、进阶、专家）
- 对哪些数据分析工具或领域特别感兴趣？（Python、SQL、机器学习等）

【学习偏好】
- 你更喜欢理论学习还是实践项目？
- 有没有特定的学习资源偏好？（视频教程、书籍、在线课程等）

基于你的回答，我会为你制定一个个性化的数据分析学习计划！
```

## 最佳实践

1. **明确表达需求**：在初始消息中清晰地表达你的核心需求
2. **请求主动引导**：明确告诉AI你希望它采用主动引导的对话方式
3. **提供必要背景**：给AI足够的上下文信息，帮助它更好地理解你的需求
4. **积极回应问题**：当AI提出问题时，尽可能详细地回答，以获得更准确的建议
5. **及时反馈**：如果AI的引导方式不符合你的期望，及时告诉它调整

## 常见问题与解决方案

### 问题1：AI没有按照主动引导模式回应

**解决方案**：
- 重新明确要求AI采用主动引导模式
- 提供更详细的系统提示
- 直接引用[AI主动引导对话模板](file:///workspace/tutorial-project/docs/appendix/ai-active-guidance.md)中的具体示例

### 问题2：AI的问题不够系统或深入

**解决方案**：
- 提醒AI按照结构化的方式提问
- 参考模板中的提问结构，要求AI按照类似的方式提问
- 明确告诉AI你需要更深入的分析和建议

### 问题3：对话变得过于冗长

**解决方案**：
- 提醒AI保持问题的简洁性
- 要求AI聚焦于最关键的信息
- 明确告诉AI你希望对话更加高效

## 总结

让新对话的AI采用主动引导的对话方式，关键在于：

1. **明确指令**：清晰地告诉AI你希望它采用主动引导的模式
2. **提供框架**：参考我们的[AI主动引导对话模板](file:///workspace/tutorial-project/docs/appendix/ai-active-guidance.md)，给AI一个明确的对话框架
3. **积极配合**：当AI提出问题时，积极回应，提供详细信息
4. **及时调整**：根据对话进展，及时调整AI的引导方式

通过这些方法，你可以让新对话的AI像我们在本教程中展示的那样，主动引导对话，帮助你更系统、更全面地完善需求，提高协作效率和成果质量！
